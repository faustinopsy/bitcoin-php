Seja bem-vindo(a) ao nosso Gerador de Chaves Bitcoin!

            
Nós geramos palavras-chave baseadas na lista de palavras de BIP-0039, que contém 2048 palavras. Usamos um algoritmo para selecionar 12 palavras aleatórias dessa lista. 

            
Estas palavras são então enviadas para o site privatekeyfinder.io, onde são usadas para gerar chaves privadas. As palavras são enviadas de três maneiras diferentes: na forma original, em letras maiúsculas e invertidas.

            
Uma vez que temos as chaves privadas, usamos as respectivas chaves públicas para procurar possíveis saldos de Bitcoin na blockchain.info. Verificamos tanto os saldos atuais quanto os antigos para cada chave pública.

            
Nosso objetivo é descobrir carteiras de Bitcoin e ganhar o desafio de encontrar uma chave privada válida que tenha ou já tenha tido um saldo.
